# Refinery Fargate Image

A re-build of [Refinery's](https://github.com/refinery/samproxy) default [Docker image](https://hub.docker.com/r/honeycombio/refinery) to use `ubuntu` instead of `scratch`.

Needed for [Refinery in Fargate Terraform module](https://github.com/vlaaaaaaad/terraform-aws-fargate-refinery#readme).
